# Java程序员面试宝典

### java的特性

封装,继承,多态，抽象

### new 和clone

1.**new** 分配内存-->调用构造函数，填充对象的各个域(初始化)--->构造方法返回对象构建完毕，引用地址发布到外部

2.**clone** 分配内存--->将原对象中的各个域填充到新对象的域---->构造方法返回对象构建完毕，引用地址发布到外部(**浅拷贝深拷贝**)         

### equals和hashCode

1. 如果两个对象相同(equls==true),那么返回的hashCode值一定要相同

2. 如果hashCode值相同,不一定相同

   **1.自反性 2.对称性 3.传递性 4.一致性**
